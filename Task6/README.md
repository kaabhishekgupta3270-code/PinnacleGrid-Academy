# Employee Management System

## Project Overview

The Employee Management System is a console-based Python application developed to manage employee records efficiently.

## Features

- Add Employee
- View All Employees
- Search Employee
- Update Employee
- Delete Employee

## Technologies Used

- Python
- Object-Oriented Programming (OOP)
- File Handling
- Exception Handling
- Modular Programming

## Project Structure

Employee_Management_System/

- main.py
- employee.py
- operations.py
- employees.txt
- README.md

## How to Run

1. Open the project folder in VS Code.
2. Open the terminal.
3. Run the following command:

```bash
python main.py
```
